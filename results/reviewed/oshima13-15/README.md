## Method

Segmentation runs on the ruler-at-top canonical orientation (see repo README,
`measure_guides.SHEET_ROTATION` and `measure_guides_v2.specimen_top`). Spots use
the reviewed detector: purple `a*-b*` two-stage guide (strong + weak, incl. the
orange-rejecting `b*<8` micro-stipple branch) as the primary
`guide_cov_pct`/`guide_present`, plus a **separate** oxidised-inclusive column
recovered only inside a confirmed purple-guide field.
